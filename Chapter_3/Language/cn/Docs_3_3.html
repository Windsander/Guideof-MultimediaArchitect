3.3 色彩的衡量

3.3 色彩的衡量

光学三要素的出现，让人们在对颜色的客观描述上，能够凭借传统色彩学体系内的参数，进行有限程度的量化。但这并不足以适用于除科学计算和测定外的批量工程作业。毕竟在算力限定的条件下，我们不可能对每一寸光的每一个物理量都进行独立的计算。同时，大量繁琐且模糊的设定也无法便于简化。

如果能够将光波本身和颜色建立起直接的可量化的转换关系，就能够解决表示上的问题了。这就是 配色函数 的由来。
